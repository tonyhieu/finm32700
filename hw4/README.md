# HFT Template Programming Homework

This scaffold contains the requested file structure and function/class signatures only.

## Structure

- `include/metaprogramming.hpp`: TMP traits and variadic/template function declarations
- `include/constexpr_math.hpp`: `constexpr` function declarations
- `include/StaticVector.hpp`: fixed-size container class skeleton
- `include/generic_algorithms.hpp`: generic algorithm declarations
- `include/OrderBookBuffer.hpp`: policy-based buffer and policy class skeletons
- `src/main.cpp`: starter usage points with TODO markers

## Build

```bash
mkdir -p build
cd build
cmake ..
cmake --build .
```
